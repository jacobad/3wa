Le html:

le HTML sert à créer le contenu du site: elle va contenir uniquement la partie textuelle du site.

le CSS:

le css va nous permettre d'ajouter à notre site: les couleurs, les formes, le fond, etc.

    Le développement
le Javascript

le javascript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page pourra varier en fonction de certains paramètres.

Le PHP

le PHP est un langage de progrmation libre, utilise pour généré des pages web dynamique, qui sont donc modifiés en fonctions de certaines informations (heures,nom de l'utlisateur,formulaire rempli par l'utillisateur,etc)

MySQL

MySQL est un système de gestion de basse de données, il permet d'enregister des infromations rempli par l'utlisateur (exemple:formulaire de creation de compte) pour ensuite être re-utilisé au moment approprié (ex:connexion à un site avec identifant et mot de passe).
